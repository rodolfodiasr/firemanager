Com base nos dados coletados, gere um ActionPlan estruturado em JSON.

Dados coletados:
{collected_data}

Dispositivo alvo:
- ID: {device_id}
- Vendor: {vendor}
- Versão: {firmware_version}

Intenção identificada: {intent}

Intenções disponíveis: create_rule, delete_rule, edit_rule, list_rules, create_group, list_nat_policies, create_nat_policy, delete_nat_policy, health_check, get_snapshot

Gere o ActionPlan no seguinte formato JSON (sem markdown):
{
  "intent": "<intent_type>",
  "device_id": "<uuid>",
  "steps": [
    {
      "sequence": 1,
      "action": "<ação específica do vendor>",
      "params": {}
    }
  ],
  "rule_spec": {
    "name": "<nome da regra>",
    "src_address": "<endereço origem>",
    "dst_address": "<endereço destino>",
    "src_zone": "<zona de origem, ex: LAN, WAN, DMZ>",
    "dst_zone": "<zona de destino, ex: LAN, WAN, DMZ>",
    "service": "<serviço/porta>",
    "action": "accept|deny",
    "comment": "<comentário>",
    "extra": {}
  },
  "nat_spec": {
    "name": "<nome da política NAT>",
    "inbound_interface": "<interface de entrada, ex: X1, X0>",
    "outbound_interface": "<interface de saída, ex: X1, X0>",
    "source": "<objeto de endereço origem ou 'Any'>",
    "translated_source": "<objeto traduzido ou 'Original'>",
    "destination": "<objeto de endereço destino ou 'Any'>",
    "translated_destination": "<objeto traduzido ou 'Original'>",
    "service": "<serviço ou 'Any'>",
    "translated_service": "<serviço traduzido ou 'Original'>",
    "comment": "<comentário>",
    "enable": true
  },
  "group_spec": null,
  "raw_intent_data": {}
}

Regras de preenchimento:
- Para create_rule ou edit_rule: preencha rule_spec, nat_spec=null
- Para create_nat_policy: preencha nat_spec, rule_spec=null
- Para delete_nat_policy: nat_spec=null, rule_spec=null, coloque rule_id em raw_intent_data
- Para create_group: preencha group_spec, rule_spec=null, nat_spec=null
- Para list_rules: rule_spec=null, nat_spec=null; coloque src_zone/dst_zone em raw_intent_data se mencionado
- Para list_nat_policies: rule_spec=null, nat_spec=null
- Para delete_rule: rule_spec=null, nat_spec=null, coloque rule_id em raw_intent_data
